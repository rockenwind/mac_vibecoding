# GitHub App 저장소 선택 UI 설계

## 목표

Repository scan 화면에서 GitHub App 설치 계정과 저장소를 선택해 스캔 입력을 자동으로 채운다.

## 범위

- 페이지 진입 시 `/api/github/installations`를 호출한다.
- GitHub App이 설정되지 않았거나 설치 목록을 가져오지 못하면 상태 메시지를 보여준다.
- 설치 계정을 선택하면 `/api/github/repositories?installationId=...`로 저장소 목록을 불러온다.
- 저장소를 선택하면 저장소 URL과 설치 ID 입력값을 자동으로 채운다.
- 기존 공개 저장소 URL 직접 입력 방식은 유지한다.

## 제외

- GitHub App 설치 화면으로 이동하는 버튼
- 저장소 검색과 필터링
- 사용자 로그인 또는 세션 관리
- 저장소 선택 이후 자동 스캔

## 오류 처리

- 설치 목록 API가 503이면 GitHub App 설정 필요 상태를 표시한다.
- 설치 목록이 비어 있으면 설치된 계정 없음 상태를 표시한다.
- 저장소 목록 API가 실패하면 저장소 목록 오류를 표시한다.
- 저장소 목록이 비어 있으면 선택 가능한 저장소 없음 상태를 표시한다.
